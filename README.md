# PID_Line_Follower_Bot
PID Line following bot using Arduino Uno
Components Used:
1) Arduino Uno(Any ESP 32 preferred due to faster memory)
2) L298N Motor Driver
3) Waveshare 5IR sensor Array module
4) 18650 Li-ion Battery
5) BO motors(preferred if you use N20 mtotrs instead)
6) Castor wheel
7) Standard yellow wheels
8) LM2596 buck converter (if u plan to use an ESP32)
